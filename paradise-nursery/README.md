# Paradise Nursery - Shopping Cart Application

## Description
Application de panier d'achat pour une boutique en ligne de plantes d'intérieur nommée **Paradise Nursery**.

Développée avec **React + Vite + Redux Toolkit** dans le cadre du cours "Developing Front-End Apps with React" (IBM - Coursera).

### Fonctionnalités
- Navigation entre Accueil, Plantes et Panier
- Affichage des plantes par catégories
- Ajout / Suppression / Modification des quantités dans le panier
- Calcul du total dynamique
- Gestion d'état avec Redux

## Technologies utilisées
- React
- Redux Toolkit
- Vite
- CSS

Projet final du cours IBM Coursera.
